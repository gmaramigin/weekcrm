Method CRM released a set of updates on April 27, 2026, focused on list screen improvements and grid fixes. The Opportunities and Activities screens now use editable grids with updated default view filters and revised button styling and positioning. The editable grid component also received an update to column width calculation logic for Date/Time fields, improving adherence to fixed-width configuration settings. On the bug-fix side, Method resolved an issue where Super Admin invitations could fail to send in some multi-entity accounts and corrected a display problem with the "make default" star icon in Manage Views on mobile devices. All updates are live for current Method users.
