Attio now allows users to request actions in plain language across notes, tasks, records, and emails. The feature carries conversational context forward, so each subsequent action builds on prior requests. Users can ask Attio to send follow-up emails, update deals, or create tasks after meetings. Before executing, the system previews the intended action and waits for explicit confirmation. The capability is available within the Attio app and directly from Slack, with automatic linking to a user's workspace via their email address.
