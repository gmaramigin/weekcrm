Cloze has unveiled Cloze Forge, a platform designed to let real estate brokerages build and deploy custom applications using AI-generated code. The platform provides a secure environment for brokerages to create tailored tools without requiring traditional development resources. Specific pricing, availability dates, and detailed technical capabilities have not yet been disclosed.
