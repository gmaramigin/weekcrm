Salesforce has released the Data 360 MCP Server in Developer Preview, a tool designed to unify enterprise data and make it accessible as context for AI agents. The server enables any agent — whether built on Salesforce or third-party platforms — to tap into consolidated data across an organization. The release is positioned as part of Salesforce's broader push to make enterprise data actionable within agentic AI workflows. Specific pricing and general availability dates have not yet been disclosed.
