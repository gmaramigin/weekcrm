Salesforce has released an open-source MCP (Model Context Protocol) Server in developer preview, enabling developers to connect AI assistants such as Cursor and Claude Code directly to Data 360 APIs using a standard stdio transport. The server provides a standardized interface for AI tools to interact with Salesforce's Data 360 platform, aimed at developers building AI-assisted workflows on top of Salesforce data. The project is available as open source and currently in developer preview status.
