Method CRM released a set of bug fixes on June 3, 2026. The update addresses an issue where the editable grid's sorting button was unexpectedly hidden on mobile devices. A fix removes Internal Note from the Quick-Add button menu, where it previously appeared as an invalid option and triggered an error. The Web To Lead form now correctly redirects users to the specified URL after submission. Additionally, the "Compare as" options in Actions now work as expected when used in criteria against date fields in the EmailsSent table. All fixes are live for current Method users.
