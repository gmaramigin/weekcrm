Method CRM released a set of bug fixes across several product areas on June 16, 2026. The update resolves a Xero integration issue where saving changes to an account triggered sync validation errors. On mobile, a fix addresses an iPhone bug that logged users out when switching apps, and another corrects a signature pop-up that failed to appear inside app ribbons on mobile devices. The report designer now retains full PDF file names in emailed reports, which were previously truncated at the first dash character. Additional fixes cover a drop-down field display issue in Tables & Fields that required a manual refresh, and a sign-in problem where browser autofill incorrectly triggered bot-detection during password resets.
