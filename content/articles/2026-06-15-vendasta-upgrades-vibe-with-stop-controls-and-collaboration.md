Vendasta has released a set of updates to Vibe, its AI-powered application builder, adding workflow controls and team collaboration features for channel partners building apps for SMB clients.

Key additions include an instant stop button that halts generation mid-run and preserves the last working preview, background chat typing so users can queue prompts during active generation, and collaborator tracking that displays each contributor's name, avatar, and timestamp on every prompt in multi-user projects. A guided Supabase secret setup now walks users through adding API keys securely, with clear error messages for missing credentials. Vibe also now supports automated verification and HTTPS certification for regional country domains such as .co.uk, .com.au, and .org.uk. Other updates include a visual edits toggle, refreshed project cards with live preview thumbnails, and leaner website cloning to reduce generation costs.
