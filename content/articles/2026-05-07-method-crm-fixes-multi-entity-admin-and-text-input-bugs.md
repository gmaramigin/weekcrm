Method CRM released a set of bug fixes on May 6, 2026, addressing issues across its multi-entity and text input features. The update resolves a problem where Super Admin invites could intermittently fail in accounts containing inactive entities. Two text input fixes were also included: one correcting an issue where auto-fill data was not recognized by screens and actions, and another eliminating a false "Field Contains Malicious Values" error that was unexpectedly triggered by certain characters.
