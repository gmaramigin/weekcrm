Method CRM released a set of bug fixes on July 2, 2026. The update resolves an issue in User Management where the "Same as Existing User" function copied only the user role without applying the source user's app and bundle permissions.

A second fix addresses the Home screen, where To-do activities were incorrectly displayed for field crew users who did not have the Activities app installed.
