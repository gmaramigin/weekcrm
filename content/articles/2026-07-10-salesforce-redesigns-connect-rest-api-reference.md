Salesforce has launched a redesigned Connect REST API reference experience aimed at helping developers locate details more quickly. The updated documentation consolidates endpoints, request and response examples, and in-line versioning into a single unified view.

The change is intended to streamline how developers interact with the Connect REST API documentation, reducing the need to navigate between separate pages for related information.
